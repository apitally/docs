## Installation

<CodeGroup>
```xml Maven
<dependency>
  <groupId>io.apitally</groupId>
  <artifactId>apitally</artifactId>
  <version>[0.1.0,)</version>
</dependency>
```

```groovy Gradle
dependencies {
  implementation 'io.apitally:apitally:+'
}
```
</CodeGroup>
