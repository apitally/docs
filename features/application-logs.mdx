The Apitally SDKs instrument various logging libraries to automatically capture logs emitted by your application during request handling.
These are associated with the corresponding request and displayed in the details modal of the request logs.

<img src="https://assets.apitally.io/docs/2025-08-26/request-log-item-logs.webp" alt="Application logs" className="rounded-xl" />

<Tip>Click on a log entry to expand it and see the full log message, in case it is truncated.</Tip>
